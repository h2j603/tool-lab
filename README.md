# color-stack

Browser-based generative poster tool in Strelka Institute visual style. Produces
print-ready SVG files suitable for import into Figma or Illustrator.

## Getting started

```bash
npm install
npm run dev          # start the web app
npm run generate     # headless Phase 1 generator
npm test             # engine unit tests
```

See `POSTER_RULES.md` for the composition philosophy. See the technical
specification in the project tracker for the implementation plan.
